# Radhe Foods

A simple local order-taking app for students and nearby homes.

## Features

- View menu (Roti, Rice, Aalu Paratha, Paneer Paratha, Sabji)
- Place orders with name, address, and phone
- Admin can view orders (future: WhatsApp notification)
- Payment: Cash on Delivery (COD)
- Mobile and laptop friendly

## Getting Started

### Backend (Node.js/Express)

1. Open terminal in `server` folder
2. Run `npm install`
3. Start server: `npm run dev` (or `npm start`)

### Frontend (React)

1. Open terminal in `client` folder
2. Run `npm install`
3. Start app: `npm start`

- Frontend: http://localhost:3000
- Backend API: http://localhost:5000

## Customization

- To add menu items, edit `server/index.js` menu array.
- For admin WhatsApp integration, see TODO in backend code.
